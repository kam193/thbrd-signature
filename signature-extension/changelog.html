Welcome in the new version of Signature sync for Gmail!

This version contains a few significant changes. See the changelog below.

Version 2.0.0 beta

  • Breaking change! Use "desktop app" oauth client type. Your OAuth credentials requires update. Please follow instructions.
  • Fix problem with sync stopped shortly after connection.
  • Show error popup after every 3 failed sync tries.
  • Show changelog page on major updates.

Version 1.0.x

  • Sync signatures for selected accounts from Gmail
  • Use your own OAuth app (web client type)

Signature sync for Gmail helps you with syncing your signatures. You can report issues on Github repo.
