Signature Sync for Gmail

Accounts to sync

Select accounts to sync and connect Gmail accounts. Synchronization is run on every startup and works in just one way - your local signature is overriden by Gmail's ones.

In case of any troubles, you will see the red annotation here. If manual sync doesn't fix it, try re-connect your Gmail account.

Google OAuth ClientID

As for now you need to create your own OAuth application in Google to make the extension working. Registring OAuth application allows you to use Google API what is necessary to sync your signatures. This is free and you can find instruction on the extension wiki. Then put the ID and secret you get in the field below.

The extension version with pre-defined OAuth client is ongoing, but registring publicly avialiable OAuth application in Google needs additional time and a few limitations that not so obvious for just a simple Thunderbird extension.

About
